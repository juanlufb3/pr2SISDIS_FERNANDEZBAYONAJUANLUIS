¿Cómo usar el simulador?

Ve al Simulador de API y pulsa cualquier botón. Spring Boot llama a Flask, captura la excepción y te muestra un mensaje claro en español. Nunca verás un stacktrace — solo mensajes entendibles para el entrenador.
